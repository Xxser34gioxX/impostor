# Impostor — juego (proyecto)

**Creator**: Sergio Lopez Feito — 2025

Pequeño proyecto React + Vite + Tailwind que implementa una réplica móvil-friendly del "juego del impostor".

Pasos rápidos (PowerShell):

```powershell
cd "C:\Users\sergi\Desktop\MisCosas\Planes y utiles\impostor"
npm install
npm run dev
```

Notas:
- Usa Tailwind CSS. Si usas VS Code instala las recomendaciones en `.vscode/extensions.json`.
- Los datos de palabras están en `src/file/words.json`. Puedes editar ese archivo para personalizar las palabras y sus categorías.
